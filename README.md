Original waste generation data obtained from Kaggle. Additional features (temperature, humidity, and holiday indicators) were incorporated for forecasting research.

Kaggle dataset Link: Mudannayake, O., Rathnayake, D., Herath, J.D., Fernando, D.K., Fernando, M., 2022a. Daily solid waste dataset. https://www.kaggle.com/datasets/ivantha/daily-solid-waste-dataset. Dataset used in “Exploring Machine Learning and Deep Learning Approaches for Multi-Step Forecasting in Municipal SolidWaste Generation,” IEEE Access, vol. 10, pp. 122570–122585, 2022.

Temperature and Humidity: https://www.wunderground.com/history/daily/lk/boralesgamuwa/date/2021-2-23

Sri Lankan Holiday: https://www.timeanddate.com/holidays/sri-lanka/
